# PolicySim.ai

PolicySim.ai is an AI-powered policy simulation platform that evaluates the
economic, environmental, and social impact of public policies before deployment.

## Use Case
Fuel tax increase combined with EV subsidy to reduce emissions and promote
sustainable transportation.

## Architecture
- Rule-based simulation engine
- Lightweight ML models for risk prediction
- Interactive Streamlit dashboard
